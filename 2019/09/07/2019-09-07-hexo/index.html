安裝 hexo 的步驟流程

參照 ref: [youtube]](https://youtu.be/jOJI9ekTzK8)

  1. 安裝Node.js

  2. 安裝hexo套件

  3. 建立資料夾

  4. 之後的文章就可以放到source/_posts裡

  5. 啟動本地端預覽

  6. 推到 github 上

  7. hexo 產上靜態網頁

  8. 修改hexo _config.yml檔，這邊要注意的是若repository是 自己第一個github.io.git，branch就用master，若不是則可以用gh-page

  9. 佈屬到 github

  10. 最後在github 上的setting裡可以找到網頁的網址

其實不只這樣，一些模板的套用修改，還需要去themes裏頭的_config.yml中做修改，或是要在網頁中加上GA或是留言版功能，
都可以再做加強，這邊只是稍微紀錄怎麼把網頁建起來
