git ssh key

add ssh key to your github

ref: github

  1. 檢查是否已經有既有之ssh key
  • 打開Git bash
  • 輸入 ls -al ~/.ssh
  • 如果有的話可能會看到
  1. 如果有ssh key的話，可以直接跳到第三步驟

  • 打開Git bash

  • 產生你的key pair

  • 預設儲存的地方在

  • 為了防止被人盜用ssh key，可再增加passphrase

  • 但為了不用每次使用ssh時都要輸入passphrase，可將ssh key加入ssh-agent中，不過要注意的是ssh-agent關掉後下次還要再重做一次

  1. 將ssh key加到github中

  • 打開Git bash

  • 複製ssh key到記事本

  • 打開github網頁

  • 點選個人圖片下方之Settings

  • 點選SSH and GPG keys

  • 點選New SSH key or Add SSH key

  • title欄位為自己註解這把ssh private用在哪裡

  • ctrl+v 到Key欄位

  • 儲存

以後當你要git push到github時，就不用再輸入帳號密碼了
